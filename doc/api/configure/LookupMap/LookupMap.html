const LookupMap(List entries, [List<LookupMap<K, V>> nestedMaps = const []])

Creates a lookup-map given a list of key-value pair entries, and optionally additional entries from other LookupMaps.

When doing a lookup, if the key is not found on entries. The lookup will be performed in reverse order of the list of nestedMaps, so a later entry for a key shadows previous entries. For example, in:

const map = const LookupMap(const [A, 1],
    const [const LookupMap(const [A, 2, B, 4]),
           const LookupMap(const [A, 3, B, 5]));

map[A] returns 1 and map[B] returns 5.

Note: in the future we expect to change entries to be a const map instead of a list of key-value pairs.
